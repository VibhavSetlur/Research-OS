# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Research OS, please do not file a public issue. Instead, please send an e-mail to the maintainer or use GitHub's private vulnerability reporting feature.

All security vulnerabilities will be promptly addressed.
